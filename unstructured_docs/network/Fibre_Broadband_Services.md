## Installation & Provisioning

### Site Survey Requirements

| Check | Purpose |
|-------|---------|
| Duct access | Confirm fibre route |
| Wayleave | Permissions if crossing third-party land |
| Power | Confirm suitable power supply |
| Rack space | For Ethernet termination equipment |
| Existing infrastructure | Assess current setup |

### Installation Process

**Standard Installation (FTTP):**
1. Order placed → Openreach survey (5 days)
2. Installation scheduled (7-14 days)
3. Engineer visit (2-4 hours)
4. Router configuration (same day)
5. Handover and testing

**Enterprise Installation (Ethernet):**
1. Order placed → Site survey (14 days)
2. Network design (7 days)
3. Wayleave/permits (14-28 days)
4. Fibre installation (7-21 days)
5. Equipment installation (1-2 days)
6. Testing and handover (1 day)
